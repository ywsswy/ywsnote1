半斤面条（小象超市陈克明手擀面）煮熟疯狂过凉水避免粘锅

多一些油（最好是调料油万能油）干辣椒爆香黑掉

炒好之后放面

调料最后（3勺酱油，1勺鸡精（可以试试多的），1勺牛肉粉（下次试试多的），1勺耗油，4勺老抽）勺是较小的勺子，最后调完大概半碗的料汁；
