|说明|-|- 种类很多，例如ETF基金，是一种在交易所上市交易的开放式证券投资基金产品，可以像股票一样交易，里面包含多支股票，所以能一定程度避免单一持股风险<br>- 场内基金：在证券交易所内交易，使用交易所的股票/证券/券商账户来买卖；<br>- 场外基金：直接向基金公司申购买卖，而不是使用交易所的股票/证券/券商账户来买卖；<br>- 需要注意的是，如果含境外股，可能也会受汇率影响；|
