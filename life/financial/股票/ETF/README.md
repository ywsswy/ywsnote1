是一种在交易所上市交易的“交易型开放式指数基金”，
结合了股票和传统基金的特点，既可以像股票一样在交易所买卖，又具有基金分散投资的优势（因为里面（“投资标的”）是投资于一篮子资产，例如多支股票、债券、商品等）

不同于LOF（“上市型开放式基金”），二者都可以场内交易，都可以申赎，但是ETF的申赎门槛高（例如需要投资者一次性买齐一篮子股票，详见PCF清单，可能一篮子最低要求对应几十万个ETF份额，动辄上百万的资金），但是套利快（申赎T+0），而LOF的申赎门槛低（投资者只需要用现金操作，基金公司可以灵活地管理现金流，例如等积攒够的时候再去买投资标的，而不需要每次申购都实时立即买入投资标的，毕竟可能投资者投入的现金连买一手都不够）申赎时间慢（T+1），所以对普通投资者而言选择LOF的申赎更像是众筹的感觉；

## 角色
- 管理人：基金公司（例如华夏、景顺长城），负责基金的投资运作，包括选股调仓；
- 托管人：银行，负责保管基金资产，监督管理人的操作；
- 普通投资者：在场内，二级市场（买卖市场）买入/卖出ETF（不影响总份额，只是换手）；
- 资金量达门槛的投资者（容易套利者）：在一级市场（净值市场）申购/赎回（影响总份额）；在溢价的时候申购（是用净值申购）然后高价卖出就可以套利；在折价的时候赎回（低价换回高净值）也可以套利；（这个机制也就导致交易价最终会趋近于实际净值）

# 风险
- 股票最糟糕的情况：公司倒闭破产清算，可能损失全部本金；
- ETF最糟糕的情况：基金清盘（当规模过小、流通性过低时，运营成本过高，基金公司可能决定清盘），但基金公司会将基金资产变现并返还给投资者。投资者的损失取决于清盘时基金资产的市场价值；

## ETF分类：
（方式1）跟踪谁
股票型ETF：跟踪股票市场指数，如沪深300ETF、标普500ETF。
债券型ETF：投资于债券市场，如国债ETF。
跨境ETF：投资于海外市场的资产，如纳斯达克100ETF。
（方式2）目标
被动型ETF：明确说明其目标是跟踪某个指数，力求复制指数的表现；
主动型ETF：会提到基金经理通过主动管理，试图超越市场基准或获取超额收益。

## 其他
- 场内基金：（可以在证券交易所）交易的基金，可以在二级市场在交易所用券商账户来买卖，ETF就属于这种；可能存在折价或溢价（相比实际价值低或高），例如境外型，大家都看好都买，但是可能由于外汇额度限制无法大量申购（例如每天限额）跟踪对应的指数就导致该基金当前的交易价格高于对应指数在美股市场的实际净值，存在风险：如果市场情绪回落或者申购大量放开，ETF的交易价格可能就会迅速跌到跟实际净值持平；另外如果境外型，可能也会受汇率影响；
- 场外基金：（要通过基金公司、银行、第三方基金销售平台、微信理财通等渠道）申购/赎回的，而不是在交易所用券商账户来买卖；常由基金经理主动运作，申购赎回需要时间，可能受额度限制（缺点，例如每天最多申购100元），存在风险：如果基金经理的投资决策失误，可能导致基金亏损；
-- （场外）公募基金：资产由独立的托管机构（通常是银行）托管，基金公司无法直接接触资金。必须定期披露基金净值（每日）、投资组合（季度）、运作情况等信息，投资者可以随时查询。
-- （场外）私募基金：私募基金的监管相对宽松，信息披露较少，风险最不确定；
- 开放式基金：份额不固定，因为可以通过灵活的申购赎回变更总份额；
- 封闭式基金：基金成立后，规模固定，有固定的存期，投资者不能随时申购或赎回，投资者只能在交易所买卖已有的份额，总份额固定；

- 场内被动型、低溢价的ETF（跨境的最好低于5%）；
