回购类型：
（1）注销式：指公司拿出一部分现金买散户手里的股票，然后把买到的注销掉。这样总股本就变少了（中性市场态度下若股价不变，总市值就会减少）。留下来的人的持股比例就变高了。
- 对外界传递的信息：管理层认为股价被低估，给大家传递信息；
- 对于坚定持有的散户是好事，因为自己持股比例就相当于变多了，后面平均每股分红就能分更多。
- 对于公司自身（控股股东），虽然控制的现金流变少了（短期看有点割肉），但是换取到的是更高的持股比例（看重未来）。
（2）库存股式：回购的部分就不叫流通股了（可以用于员工股权激励；常见误区：以为公司低价卖，先屯着等高价的时候再卖掉可以赚“现金流”收益；但是通常市场不允许这么玩，毕竟公司&内部人是信息优势方，公平起见不能在二级市场跟散户对赌）；
所以要区分开几个角色：
- 公司法人：偏虚拟角色，回购花的钱是从这里出的，不允许低买高卖；
- 控股股东：持股比例足够大，对公司有一定控制决策权，这里才是确切的人，实体的股东，那么他个人是可以低买高卖的，不过要遵守时间窗口要求（例如必须6个月）
- 外部股东：公众股东散户；


总结注销式是"我给股东钱"，库存股式是"我先囤着，以后再说"，前者诚意更足，后者工具属性更强
