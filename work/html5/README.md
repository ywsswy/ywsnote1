## html5大纲算法（区块/盒子+标题/标签 自成目录）
https://gsnedders.html5.org/outliner/
《深入理解HTML5：语义、标准与样式》
1. 只用h1
2. html/body是最外层区块，每个区块的标题要看他里面的hx
3. 有区块没标题时：取默认标题是untitiled
4. 每一个标题都会自动创建一个区块，（区块间并列/内嵌，取决于标题间的关系）
5. div与是否创建区块无关
