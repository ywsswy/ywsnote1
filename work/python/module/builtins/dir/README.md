内建的dir函数来列出模块定义的标识符。标识符有函数、类和变量
当你为dir()提供一个模块名的时候，它返回模块定义的名称列表。如果不提供参数，它返回当 前模块中定义的名称列表
import sys
dir(sys)
dir()
