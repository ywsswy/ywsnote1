## 1.道路交通标志和标线
```
255种道路交通标志，72种道路交通标线
道路交通标志(2)：主标志、辅助标志
主标志(6)：禁令标志（一般为红圈）、指示标志（蓝底,指示行进）、警告标志（黄底,注意危险）、指路标志（蓝底 or 绿底,传递道路方向、地点、距离信息）、旅游区标志、告示标志
道路交通标线（3）：禁止标线、指示标线、警告标线
```
