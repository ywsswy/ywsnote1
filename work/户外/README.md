## 木材燃烧和木煤气燃烧不同：
木材燃烧（火焰温度是600-1600摄氏度），C6H10O5 + 6O2 → 6CO2 + 5H2O + 热量
但是同时，木材加热至高温（800摄氏度左右）时，木材中的碳、氢、氧等元素发生【热解反应】/气化反应？，产生一种可燃性气体混合物，即木煤气。
木煤气的生成：C6H10O5 → 5CO + 5H2 + 2CH4 + 热量
