# Ghost版
是在正版Windows基础上修改编辑过的系统镜像。存在许多不可预知或者潜在的风险;
Ghost软件是一种磁盘克隆工具，它可以将整个硬盘或分区的内容复制到另一个硬盘或分区中，包括操作系统、应用程序和数据等；


系统升级会产生windows.old文件夹，可以在 系统-》存储-》中清理

alt+tab 出现多个浏览器窗口：系统->多任务处理->不显示选项卡
