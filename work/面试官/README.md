## 先预约阶段
介绍面试阶段，介绍部门，问意愿（工作地、部门）

## 准备阶段
- 看简历（1）确定语言，提前准备编程题；（2）准备提问（语言+组件+项目）；

## 面试阶段
面试官介绍，确认base、入职时间、编程语言（愿意转团队方向吗）、专业匹配度（数据结构、操作系统）

聊兴趣，主动钻研精神；（硬件、光刻机）

听项目（逻辑表达）
- STAR模型，看过往经历，通过追问细节补全：
-- S-场景-为什么做；【】
-- T-目标-要做成什么、规模：数据量，用户数量；
-- A-行动-哪些执行步骤以及如何执行（拒绝模糊）；【】挑战设计、问创新性、协作性；
-- R-结果-是否符合目标。

项目中使用的某个组件的理解、原理；
某个方向的具体选型、行业最新进展；
挑战点（遇到问题解决问题）、性能问题难点、亮点、创新点；
复盘：自己认为有什么问题；

语言基础知识；

AI的使用多吗，热门技术的理解；对AI的看法（边界、哪些行业替代不了）

写代码（编程题库）；
可以先讲思路、分析时间复杂度；
