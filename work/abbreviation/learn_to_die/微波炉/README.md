产生交变电场；
极性分子（主要是水）就会摩擦生热；

穿透性：
金属：无法穿透（因为金属会产生相反的场抵消掉）
陶瓷/水：大部分能穿透；穿透深度2~3cm，再深的部分只能靠热传导/对流加热；


危险性：
（1）不能热密封的食物，例如鸡蛋带不带壳都有密封腔都不行；
（2）金属不行，会起火、特殊的不锈钢只能“短暂”加热；
（3）只能特定（高温烧制明确写着可微波）的陶瓷才行；
（4）不能光滑的容器煮纯水，看似平静有可能一碰就炸沸，必须里面有东西或者不光滑；

煮面是否需要盖盖子：
实践下来不盖也行，不能太满，留30%的空间；
可以200度，煮两个90秒；（如果是用锅煮，要煮至少90秒）
